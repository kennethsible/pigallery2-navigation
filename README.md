# PiGallery2 Navigation Userscript

[![Static Badge](https://img.shields.io/badge/install%20userscript%20-%20pigallery2%20navigation%20-%20blue?style=for-the-badge)](https://github.com/kennethsible/pigallery2-navigation/raw/refs/heads/main/pigallery2-nav.user.js)

In PiGallery2, you can navigate through photos using the side buttons or arrow keys. However, on mobile devices, the side buttons disappear when a photo fills the screen. This means users have to tap twice: once to make the buttons reappear, and again to go to switch photos. To make mobile navigation more intuitive, this userscript allows you to navigate by simply tapping either side of the screen.

> [!IMPORTANT]
> You will need to install a userscript manager for your browser, such as [Tampermonkey](https://github.com/Tampermonkey/tampermonkey) or [Userscripts](https://github.com/quoid/userscripts).
